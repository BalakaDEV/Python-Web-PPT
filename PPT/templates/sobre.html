Sobre o projeto

O projeto PPT foi feito visando a criação de uma pagina usando Flask e Python.
A ideia inicial do projeto era sair da criação de executaveis em Python (que na maioria das vezes contavam como virus)
e colocar toda a aplicação a acesso de qualquer dispositivo e qualquer pessoa.
Assim trazendo aplicações Python sem precisar baixar absolutamente nada graças a biblioteca que integra Python na Web.
Apenas lembrando que o projeto não é de extrema qualidade pois foi feito com o intuito de estudar novas bibliotecas e possibilidades
Espero que se divirta obrigado por testar!

Source do projeto para Download: https://github.com/BalakaDEV/Python-Web-PPT
